# ComputerVision

This repository contains some of my computer vision projects.

## Projects

### Circle Detector (Hough Transform)

- **Location**: [circle_detector/](circle_detector/)
- **Description**: Implementation of the Day 2 Assignment for detecting circles in images using OpenCV's Hough Circle Transform
- **Features**:
  - Robust preprocessing pipeline (grayscale, blur, histogram equalization)
  - Configurable Hough Circle Transform parameters
  - Visual annotation with green outlines and red centers
  - Comprehensive statistics reporting
  - CLI interface with extensive parameter tuning
  - Tested on 4 different image scenarios (coins, bubbles, challenging/noisy, real-world scenes)

### Pencil Sketch Converter

- **Location**: [pencil_sketch_converter/](pencil_sketch_converter/)
- **Description**: Converts photos into pencil sketch style images
